📋 Sobre o Projeto

Este projeto foi desenvolvido para centralizar informações sobre eventos tecnológicos gratuitos realizados em Brasília durante o ano de 2026, facilitando o acesso de profissionais, estudantes e entusiastas de tecnologia.
